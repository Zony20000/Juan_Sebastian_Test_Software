# Taller de Pruebas 2

Dentro de este repositorio se encuentra el codigo y pruebas de los mismos en base a los dado en el archivo  taller2.html que esta cargado tambien, en base a esto se tuvo que hacer unas modificaciones para el ultimo punto esto, por que el codigo dado no se mostraba el relacionado con este estudiante por ello se realizo un random.randint dentor de un rango de 1 a 4.

## Set-Up
Para la ejecuccion de este codigo de forma correcta y igual al estudiante es necesario tener los siguentes atributos dentro del VENV o entorno virtual:

* version de python 3.13.5.
* include-system-site-packages = false.
* Las carpetas App y Test solo se agregan dentro del venv, estas no deberan estar dentro de otras a excepto que requiera cambiar las importaciones.

Con esto aspectos tendra el espacio perfecto para la ejecuccion del software y pruebas.
